## ✨ 核心功能

### 1. 用户认证
- **学号登录**：12位学号（如：202205570603）
- **用户注册**：支持新用户自助注册
- **数据隔离**：每个用户独立的数据空间

### 2. 智能事件提取
- **文本识别**：从通知文本自动提取事件信息
- **多字段识别**：时间、地点、主办单位、活动类型等15+字段
- **循环事件**：支持"每周一"等周期性提醒
- **截止时间**：自动识别"3月15日前"等截止提醒

### 3. 地点标准化
- **智能匹配**：别名自动转换（"信科北501" → "信息科技大楼北501"）
- **湘大地点库**：预置常用地点
- **校验提醒**：非校内地点提示确认

### 4. 冲突检测
- **事件冲突**：同时间多个事件标黄警告
- **课表冲突**：与课程时间冲突标红强烈提醒

### 5. 多格式上传
- **图片识别**：OCR自动提取文字
- **PDF解析**：自动读取文本
- **Word文档**：.docx格式支持
- **网页爬取**：粘贴URL自动提取
- **批量导入**：多条通知自动拆分

### 6. 课表管理
- **可视化**：7×5课表网格
- **冲突检测**：新建事件自动检测课程冲突

### 7. 历史归档
- **自动归档**：已完成事务归档
- **搜索**：按标题、地点搜索
- **导出**：CSV格式导出

## 🔧 故障排查

### 问题1：一键部署失败

**检查Python**：
```bash
python --version
# 应显示 Python 3.8 或更高版本
```

**检查MySQL**：
```bash
mysql --version
# 应显示 MySQL 8.0 或更高版本
```

### 问题2：数据库初始化失败

**错误提示：Access denied**
- 原因：MySQL密码错误
- 解决：确认输入正确的root密码

**错误提示：Unknown database**
- 原因：首次安装MySQL
- 解决：脚本会自动创建数据库

**错误提示：字符编码错误**
- 原因：MySQL默认字符集不是utf8mb4
- 解决：
```bash
mysql -u root -p --default-character-set=utf8mb4 < init_database.sql
```

### 问题3：后端启动失败

**ModuleNotFoundError: No module named 'xxx'**
- 原因：依赖未安装
- 解决：
```bash
cd backend
venv\Scripts\activate.bat
pip install -r requirements.txt
```

**pymysql.err.OperationalError: Can't connect**
- 原因：.env文件配置错误
- 解决：检查 `backend\.env` 文件中的数据库配置

### 问题4：前端无法访问

**端口被占用**
- 解决：修改启动脚本中的端口号（默认8080）

**登录后提示网络错误**
- 检查后端是否正常运行（应在5000端口）
- 检查浏览器控制台是否有CORS错误

### 问题5：OCR功能不可用

**原因**：未安装Tesseract OCR引擎

**解决**（可选）：
- Windows：https://github.com/UB-Mannheim/tesseract/wiki
- 安装后添加到环境变量

## 📊 数据库管理

### 查看用户
```sql
USE campus_reminder_system;
SELECT * FROM users;
```

### 查看事件
```sql
SELECT * FROM text_events WHERE user_id = 2;
```

### 重置用户密码
```sql
UPDATE users 
SET password_hash = MD5('新密码') 
WHERE student_id = '学号';
```

### 清空测试数据
```sql
DELETE FROM text_events WHERE user_id = 2;
DELETE FROM reminder_tasks WHERE user_id = 2;
```

### 备份数据库
```bash
mysqldump -u root -p campus_reminder_system > backup.sql
```

### 恢复数据库
```bash
mysql -u root -p campus_reminder_system < backup.sql
```
